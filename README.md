# OS Scheduling

This repository includes implementations and simulations
of various scheduling algorithms. The list of current implementations is shown below.

**Process scheduling**

- FCFS
- SJF
- Priority
- Round-Robin

**Memory scheduling**

- FIFO
- OPT
- LRU

**Disk Scheduling**

- FCFS
- SSTF
- LOOK
- CLOOK
